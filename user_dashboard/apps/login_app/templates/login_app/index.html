Welcome to User Message Board

This project demonstrates login/registration, password handling, user profile editing, and playing with DateTime. Mostly to customize the output of message timestamps. Powered by Python, Django, SQLite, and Bootstrap.

Click the start button to register.

User Dashboard

The dashboard shows a list of all users and their user level. You can view a user's profile by clicking their name and can leave a message for them or comment on their other messages.

Leave Messages and Comments

Users will be able to leave a message to another user using the application. They can also comment on their own and other user's messages.

Edit User Information

Admins will be able to edit another user's information (email address, first name, last name, etc.) Each user can also manage their own profile and change various aspects of their information.
